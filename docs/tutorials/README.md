# MedAnki Tutorials

Step-by-step guides to help you get the most out of MedAnki.

## Beginner Tutorials

### [Create Your First Deck](./first-deck.md)
**Time:** 10 minutes | **Difficulty:** Beginner

Learn the basics by creating a flashcard deck from a sample PDF. Covers uploading, configuration, generation, and importing to Anki.

## Intermediate Tutorials

### [Batch Processing](./batch-processing.md)
**Time:** 15 minutes | **Difficulty:** Intermediate

Process multiple files efficiently using the CLI. Learn to automate deck creation and organize large collections.

### [Optimizing Your Cards](./optimizing-cards.md)
**Time:** 20 minutes | **Difficulty:** Intermediate

Review and improve generated cards in Anki. Learn editing techniques, suspension strategies, and quality control.
